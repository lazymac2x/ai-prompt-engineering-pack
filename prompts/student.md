# Student & Learner Prompt Pack (15 Prompts)

> Prompts designed for students, researchers, and lifelong learners.
> Replace `{placeholders}` with your own context before use.

---

## 1. Research Paper Deep Dive

```
Help me deeply understand the following research paper/article.

Title: {paper_title}
Field: {subject_area}
Key content (abstract or summary):
{paste_abstract_or_summary}

Please provide:
1. Plain-language summary (explain it like I'm a smart undergraduate)
2. Key findings and why they matter in the real world
3. Methodology breakdown — what did they actually do?
4. Strengths and weaknesses of the research
5. How this connects to broader themes in {subject_area}
6. 5 follow-up questions this paper raises
7. If I wanted to cite this paper in my own work on {my_topic}, what angle would be most useful?
8. 3 related papers or concepts I should look into next
```

## 2. Essay Writing Assistant

```
Help me write a {essay_type — e.g., "argumentative", "analytical", "compare and contrast", "narrative"} essay.

Topic: {topic}
Thesis statement (if I have one): {thesis_or_leave_blank}
Course: {course_name}
Academic level: {high_school/undergraduate/graduate}
Required length: {word_count}
Sources I must use: {list_sources_if_any}
Formatting style: {apa/mla/chicago}

Please:
1. If my thesis is weak or missing, suggest 3 strong thesis options
2. Create a detailed outline with:
   - Introduction with hook strategy
   - Body paragraphs (topic sentence, evidence, analysis, transition for each)
   - Counter-argument section (if argumentative)
   - Conclusion that goes beyond restating the thesis
3. Write the introduction and one body paragraph as a model
4. Identify potential logical fallacies to avoid
5. Suggest transition phrases that fit this essay type
6. Provide a checklist for self-editing before submission
```

## 3. Personalized Study Plan

```
Create a study plan for {subject_or_exam}.

My situation:
- Goal: {pass_exam/master_subject/improve_grade}
- Current knowledge level: {beginner/intermediate/advanced}
- Exam/deadline date: {date}
- Hours I can study per day: {hours}
- Days per week available: {days}
- Learning style preference: {visual/auditory/reading/hands-on}
- Biggest struggle areas: {list_weak_topics}
- Resources I have access to: {textbook, lecture notes, online courses, etc.}

Please provide:
1. Week-by-week study schedule with specific topics per session
2. Study technique recommendations for each topic type (memorization vs. problem-solving vs. conceptual)
3. Active recall questions for each major topic
4. Spaced repetition schedule for key facts
5. Practice test strategy (when to start, how many, how to review)
6. Break and rest schedule (Pomodoro or other technique)
7. Warning signs that I need to adjust the plan
8. The "minimum viable" version of this plan if life gets busy
```

## 4. Concept Explainer (Feynman Technique)

```
Explain {concept} to me using the Feynman Technique.

Subject: {subject_area}
My current understanding: {what_I_think_I_know}
What confuses me specifically: {confusion_points}
My level: {high_school/undergrad/grad/professional}

Please:
1. Explain the concept in the simplest possible language (no jargon)
2. Use a real-world analogy that makes the core idea click
3. Build up from the simple explanation to the full complexity, step by step
4. Highlight the one key insight that makes everything else fall into place
5. Point out the most common misconceptions and why they're wrong
6. Give me 3 "test yourself" questions — if I can answer these, I truly understand it
7. Show how this concept connects to {related_topic_I_already_understand}
8. Provide a one-paragraph summary I could use to explain this to someone else
```

## 5. Exam Preparation & Practice

```
Help me prepare for an exam on {subject}.

Exam format: {multiple_choice/short_answer/essay/mixed/problem_solving}
Topics covered:
{list_all_topics}

Key topics I feel weakest on:
{list_weak_areas}

Textbook/materials used: {references}

Please:
1. Create a topic-by-topic study priority list (high/medium/low) based on typical exam weighting
2. Generate {number} practice questions that match the exam format:
   - {number} easy (foundational concepts)
   - {number} medium (application and analysis)
   - {number} hard (synthesis and edge cases)
3. Provide detailed answer explanations (not just the answer, but WHY)
4. Create a "cheat sheet" — the essential formulas, dates, or concepts on one page
5. Common exam traps and tricks to watch out for
6. Time management strategy for the exam itself
7. Night-before and morning-of recommendations
```

## 6. Note Summarization & Organization

```
Organize and summarize my messy notes into a clear study resource.

Subject: {subject}
Topic: {specific_topic}
My raw notes:
{paste_notes_here}

Please:
1. Organize these notes into a logical hierarchy with clear headings
2. Fill in any obvious gaps in the content (mark additions with [ADDED])
3. Create a one-page summary of the most important points
4. Convert key relationships into a concept map (text-based)
5. Highlight definitions and key terms in a glossary format
6. Generate 10 flashcard pairs (front: question, back: answer) from these notes
7. Identify areas where my notes are unclear or potentially incorrect — flag with [VERIFY]
8. Add mnemonic devices for lists or sequences that need memorization
```

## 7. Math & Science Problem Solver

```
Help me solve and understand this {math/physics/chemistry/etc.} problem.

Problem:
{paste_problem}

What I've tried so far:
{my_attempt_or_where_I_got_stuck}

My level: {course_name_or_level}

Please:
1. Do NOT jump to the answer. First, explain the approach — what strategy applies and why
2. Identify which formulas, theorems, or principles are needed
3. Solve step by step, explaining the REASONING behind each step (not just the algebra)
4. At each decision point, explain why you chose that path over alternatives
5. Box the final answer with units
6. Show me a common mistake students make on this type of problem
7. Give me a similar problem (with answer) so I can practice the same technique
8. Explain when I would and wouldn't use this approach in the real world
```

## 8. Literature Analysis

```
Help me analyze {literary_work — title and author} for my {course_name} class.

Specific focus: {essay_topic_or_theme_to_analyze}
Passages I find important:
{paste_key_passages_or_page_numbers}

Please provide:
1. Overview of major themes and how they develop throughout the work
2. Character analysis for {character_names} — motivations, arc, symbolic role
3. Analysis of the specific passages I've highlighted — literary devices, subtext, context
4. How the author's biographical or historical context influences the work
5. Comparison with {another_work_or_movement} if relevant
6. 3 strong thesis statements I could argue in an essay
7. Key quotes with page numbers and analysis for each thesis
8. Critical perspectives (feminist, postcolonial, Marxist, psychoanalytic) and which fits best
9. Counter-arguments to anticipate for each thesis
```

## 9. Language Learning Accelerator

```
Help me practice and improve my {target_language}.

My current level: {beginner/intermediate/advanced} (or {CEFR_level})
Native language: {native_language}
Goal: {conversational_fluency/business_proficiency/exam_prep/travel}
Specific topic to practice: {topic — e.g., "ordering food", "job interview", "debating politics"}

Please:
1. Write a sample dialogue on this topic (with translation)
2. Extract key vocabulary (15 words/phrases) with:
   - Pronunciation guide
   - Example sentence
   - Common mistakes native speakers of {native_language} make with this word
3. Grammar point focus: explain one grammar rule that appears in the dialogue
4. Give me 5 sentences to translate from {native_language} to {target_language} (progressive difficulty)
5. Cultural context: what would a native speaker do differently in this situation?
6. Listening/speaking exercise suggestion
7. Memory trick for the 3 hardest vocabulary words
```

## 12. Critical Thinking & Argument Analysis

```
Help me critically analyze the following argument/claim:

Claim: {statement_or_argument}
Source: {where_it_comes_from}
Context: {course_or_discussion_context}

Please:
1. Identify the argument structure (premise > premise > conclusion)
2. Classify the type of reasoning (deductive, inductive, abductive)
3. Check for logical fallacies — name each one found and explain why it's a fallacy
4. Evaluate the evidence: is it sufficient? Relevant? Current?
5. Present the strongest steelman version of this argument
6. Present the strongest counter-argument
7. Identify hidden assumptions the argument relies on
8. Rate the overall strength of the argument (weak/moderate/strong) with justification
9. Suggest what additional evidence would make this argument more convincing
```

## 13. Presentation Builder

```
Help me create a presentation for {course_or_event}.

Topic: {topic}
Duration: {minutes}
Audience: {classmates/professors/conference}
Format: {in_person/virtual}
Slide count target: {number}
Must include: {specific_requirements}
Visual aids available: {charts, demos, videos, etc.}

Please provide:
1. Presentation outline with slide-by-slide content:
   - Slide title
   - Key points (bullet points, max 3-4 per slide)
   - Speaker notes (what to SAY, not what's on the slide)
2. Opening hook that grabs attention in the first 30 seconds
3. Transition sentences between major sections
4. A story or example that makes the core concept memorable
5. Audience interaction moment (question, poll, activity)
6. Strong closing statement and call to action
7. Anticipated Q&A questions with prepared answers (5 questions)
8. Timing guide: how many minutes per section
```

## 14. Scholarship / Application Essay

```
Help me write a compelling application essay.

Application for: {scholarship_name/university/program}
Essay prompt: {paste_exact_prompt}
Word limit: {limit}
My background: {relevant_personal_background}
Key experience I want to highlight: {experience}
Why this opportunity: {why_this_specific_program}
My career goal: {goal}

Please:
1. Brainstorm 3 different angles/narratives I could take
2. For the strongest angle, create a detailed outline
3. Write a compelling opening paragraph (show, don't tell)
4. Suggest a narrative arc (setup > tension > resolution > insight)
5. Help me connect my personal story to the program's values
6. Review for: cliches to avoid, generic statements to make specific, showing vs. telling
7. Provide a checklist: Does it answer the prompt? Is it within word count? Is it memorable?
8. Write the closing paragraph that ties everything together
```

## 15. Self-Directed Learning Curriculum

```
Design a self-directed learning curriculum for {skill_or_subject}.

My starting point: {current_knowledge}
Target proficiency: {what_I_want_to_be_able_to_do}
Time commitment: {hours_per_week} for {duration_weeks_or_months}
Budget for resources: {budget}
Preferred learning style: {reading/video/projects/interactive}

Please provide:
1. Learning roadmap divided into phases (Foundations > Intermediate > Advanced > Mastery)
2. For each phase:
   - Specific topics to cover in order
   - Best free resource for each topic (with URLs if known)
   - Best paid resource if budget allows
   - Project or exercise to prove mastery before moving on
3. Milestones: "You've completed this phase when you can..."
4. Community recommendations (forums, Discord servers, meetups)
5. Portfolio / proof-of-learning suggestions
6. Common plateaus learners hit and how to push through them
7. How to know when you've reached your target proficiency
```
